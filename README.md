# my-computer-vision
This project is aimed at learning computer vision techniques and applications.

## Features
- Implementation of basic computer vision algorithms
- Tutorials on using OpenCV

## Technologies Used
- Python
- OpenCV
- NumPy

## Installation
1. Clone the repository: `git clone <repository-url>`
2. Install required packages: `pip install -r requirements.txt`
